GlassHopper | Bar Crawl Like a Champion.

A Dev Bootcamp Final Project January 2015

GlassHopper is a mobile application that gives you a statistical snapshot of the current state of bars around you via geolocation. Get realtime data such as girl to guy ratio, number of people, noise level, and more!

Backend Tech Stack: NodeJS, Express, MongoDB, Mongoose

Client Tech Stack: AngularJS, Ionic + Cordova

Team Members: Stefano Schiavi, Neil Lambert, Sharmila Jesupaul

Project Goals: Learn an entirely new stack of languages, github workflow, product development cycle, and what it takes to be a dev team leader.

Most Impressive Technical Feats: Geospatial database queries, JWT Token Authentication

Issues/Bugs Remaining: Proper Google map size rendering, Fat controllers, few client/server side validations, average age calculation, testing

Hours Spent: ~200 HRS per person

Lessons Learned: Integrate testing early to avoid headaches later, don't install application wide technical infrastructure without thinking through its less obvious implications, timebox features and stick to them

Circle | Welcome Home.

http://p10.herokuapp.com December 2014

A private social network for you and your family. Securely share posts, polls, and photos between your family members.

Tech Stack: Ruby on Rails, Postgresql

Project Goals: Build something beautiful with a focus on authentication and Activerecord associations in 7 days.

Most Impressive Technical Feats: Three-layer authentication, self-referential polymorphic association, Amazon S3 integration, indexing entire database via tags for inhouse search engine

Issues/Bugs Remaining: FBOauth still needs app approved ID, Private Message and Inbox unfinished, Album CSS styling unfinished

Hours Spent: ~85 HRS

Lessons Learned: Prioritize features and properly allocate timeboxes, sleep is necessary
