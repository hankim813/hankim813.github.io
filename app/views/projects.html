A mobile application that gives you a statistical snapshot of the current state of bars around you via geolocation. Get realtime data such as girl to guy ratio, number of people, noise level, and more!

Backend Tech Stack: NodeJS, Express, MongoDB, Mongoose

Client Tech Stack: AngularJS, Ionic + Cordova

Team Members: Stefano Schiavi, Neil Lambert, Sharmila Jesupaul

Project Goals: Learn an entirely new stack of languages, github workflow, product development cycle, and what it takes to be a dev team leader and a product manager

Most Impressive Technical Feats: Geospatial database queries, JWT Token Authentication

Hours Spent: ~200 HRS per person
